Growth Studio
===

> The Demo Code of Book "Growth Enginner: Thinking Web", and for Growth Studio Team.

Setup
---

``Requirements``: Python 3.5

1. Installl Fabric 3

```bash
pip install fabric3
````

2. Install Requirements.txt

```bash
fab install:dev
```

3. Install geckodriver

```bash
brew install geckodriver
```

4. Deploy

```bash
fab deploy
```

Run Test

```
fab test
```
